\# SUV Purchase Prediction



This project builds a logistic regression pipeline to predict whether a user purchases an SUV based on the given dataset.



\## Project Structure

\- data/ - dataset files

\- src/ - modular Python source files

\- notebooks/ - Jupyter notebook version of the workflow

\- outputs/ - generated evaluation plots

\- main.py - entry point for running the pipeline



\## Features

\- Data loading and preprocessing

\- Logistic Regression model training

\- Prediction and evaluation

\- Confusion matrix visualization

\- Decision boundary visualization



\## Requirements

Install dependencies using:



pip install -r requirements.txt



\## How to Run

Run the main script:



python main.py



\## Output Files

Generated plots are stored in the outputs/ folder:

\- confusion\_matrix.png

\- decision\_boundary.png
